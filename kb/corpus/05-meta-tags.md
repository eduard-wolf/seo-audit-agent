# Title Tags and Meta Descriptions

## Title Tag (`<title>`)

The title tag is the most important on-page SEO element. It appears as the clickable headline in SERPs and browser tabs.

### Recommended length

50–60 characters (Google typically truncates beyond ~580 px, roughly 60 chars).

### Best practices

- Place the primary keyword near the beginning.
- Write a unique title for every page.
- Include the brand name at the end, separated by a pipe or dash: `Primary Keyword | Brand Name`.
- Avoid keyword stuffing — write for humans first.

## Meta Description

The meta description does **not** directly influence rankings, but it affects click-through rate (CTR), which is a strong indirect signal.

### Recommended length

140–160 characters. Google sometimes auto-generates descriptions from page content when the provided description is considered low-quality.

### Best practices

- Write a compelling call to action.
- Include the target keyword naturally (Google bolds matched terms in the snippet).
- Every page should have a unique description.
- Do not duplicate title content in the description.

## Duplicate Titles and Descriptions

Duplicate `<title>` or `<meta name="description">` across multiple pages wastes the differentiation opportunity and can confuse crawlers about which page to rank for a given query.

## Open Graph and Twitter Cards

While not a direct ranking factor, `og:title`, `og:description`, and `og:image` determine how content appears when shared on social media — affecting indirect traffic and brand perception.
